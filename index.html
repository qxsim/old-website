Messaging System

Designed and built a multi-threaded, fully-featured messaging system.
Used various methods to deliver a coherent solution, including using concurrency.
Features include login system, group chats, message traversal and deletion, message storage.

Message Decryptor

Designed and built a message decryption system, with a fast and fluid GUI.
Offered multiple means of decryption using different algorithms, that solve the problem with varying degrees of speed and accuracy, with each offering its own costs and benefits.
Developed GUI from scratch using the widely-used MVC model.

Word Prediction System

Designed and built a word prediction system, using a predefined dictionary of words, and constructing a trie data structure to hold 250,000+ words.
Designed efficient and powerful algorithms to traverse this vast data structure, to calculate a word prediction, given a specific input representing a prefix.
Used J-Unit testing to ensure correctness of all the algorithms used in the solution.

Robot Warehouse

Worked as part of a small team to design and build a large-scale multi-robot system that can complete a designated set of given jobs in the most efficient way, maximising productivity.
Various highly advanced AI methods were used such as A* search, Bayesian Networks etc, in order to aid the robots in selecting and distributing jobs to complete the given set in the shortest amount of time.
Implemented communication methods, so the robots were able to ‘speak’ to other robots, helping them all keep track of current tasks, and to select jobs based on what other robots are currently doing, as well as, keeping out of the way of other robots to avoid collisions.
