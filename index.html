Sentiment Analysis on Tweets Using Document Embeddings

  • This is my final-year project (dissertation) which was an investigation into the various methods I could use to classify tweets by their sentiment.
  • I trained word vector-based, and paragraph vector-based models on a dataset consisting of 1.6 million tweets, in conjunction with various classifiers in order to find the best performing method in which to obtain the calculated sentiment of unseen tweets.
  • I designed and implemented all the various document representations, as well as the classifiers which included Logistic Regression, Cosine Similarity, Linear Discriminant Analysis, and multiple different forms of Neural Networks.
  • In the best case, I obtained an accuracy of 79.6%, using a very specific document vector, coupled with a Neural Network classifier.
